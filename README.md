# DualSync-Game
🗲 DUALSYNC: Neural Sync Protocol
DualSync es un inhibidor de distracción interactivo diseñado para entrenar la coordinación bimanual y la atención dividida. Ambientado en una estética ciberpunk minimalista, el sistema desafía al usuario a sincronizar ambos hemisferios cerebrales mediante el control de dos entidades independientes.
🎮 Características Principales
⦁	Frecuencia de Enlace 432Hz: Audio ambiental diseñado para fomentar la concentración profunda.
⦁	Protocolos de Entrenamiento: Tres niveles de dificultad (I, II, III) con curvas de aprendizaje recalibradas para una experiencia justa pero exigente.
⦁	Motor de Partículas: Efectos visuales fluidos mediante HTML5 Canvas.
⦁	Diseño Responsivo: Compatible con dispositivos táctiles y punteros de alta precisión.
🚀 Instalación y Uso
No requiere dependencias externas ni compilación.
1.	Clona este repositorio:
2.	git clone [https://github.com/TU_USUARIO/dualsync-game.git](https://github.com/TU_USUARIO/dualsync-game.git) 
3.	Abre index.html en cualquier navegador moderno.
🛠️ Ajustes Técnicos (V2.0)
En la última versión, se optimizaron los Protocolos II y III respondiendo al feedback de la comunidad:
⦁	Velocidad de Entrada: Reducida en un 30% en niveles avanzados para mejorar el tiempo de reacción inicial.
⦁	Crecimiento Orgánico: La aceleración de los obstáculos ahora escala de forma logarítmica según el puntaje obtenido.
⦁	Hitboxes: Ajuste de colisiones para permitir movimientos más fluidos.
